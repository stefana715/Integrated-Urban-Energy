# Figure directory — Paper 3

All 14 publication figures are generated by `code/postprocessing/generate_all_figures.py`
using the unified style constants in `code/postprocessing/style.py`.

Re-generate all figures:
```bash
python code/postprocessing/generate_all_figures.py
```

## Style

- Font: Arial, 9/10/11 pt (tick/label/title)
- DPI: 300 (print), 150 (screen)
- Width: 7.1" (2-column) or 3.5" (single-column)
- Color palettes: see `code/postprocessing/style.py`
